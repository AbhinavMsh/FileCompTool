## How to run locally

1. Clone the repository
```bash
git clone https://github.com/AbhinavMsh/FileCompTool.git
cd FileCompTool
```

2. (Optional) Create and activate a Python virtual environment
```bash
# macOS / Linux
python3 -m venv venv
source venv/bin/activate

# Windows (cmd)
python -m venv venv
venv\Scripts\activate

# Windows (PowerShell)
.\venv\Scripts\Activate.ps1
```

3. Install dependencies (if using the Flask backend)
```bash
pip install -r requirements.txt
```

4. Run the Flask app (optional)
```bash
python app.py
```
